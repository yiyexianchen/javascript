    回顾

    null instanceof Object 是flase --null的类型是object 它不是由object创建 就是空的--
    undefined instanceof Object 是flase --undefined的类型是undefined 跟object没有半毛钱关系--
    null===undefined 是flase
    null==undefined 是true
    NaN==NaN 是flase--NaN和任何数据比较 永远返回false 和任何数字计算返回NaN 判断NaN用isNaN(x)--
    str>10000 不管数字多大 是flase--关系运算中字符创转为数字 转不成 为NaN--
